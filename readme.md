## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Install this project using Composer to prevent any funny business.
 - PHP 7.0+
 - MariaDB 5.5+
 - At least 256MB of memory allocated to PHP
 - At least 200MB of free disk space

More information is available on the [CraftCMS website](https://docs.craftcms.com/v3/requirements.html "CraftCMS Server Requirements").
